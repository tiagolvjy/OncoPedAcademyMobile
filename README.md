# [PT]
# OncoPed Academy — Mobile

Plataforma educacional mobile voltada para estudantes de medicina e profissionais da saúde, com foco em oncologia pediátrica. Desenvolvida como Trabalho de Conclusão de Curso (TCC) no Centro Universitário CESMAC, em parceria com o Centro de Inovação Tecnológica do Cesmac - CITEC.

![React Native](https://img.shields.io/badge/React_Native-20232A?style=flat&logo=react&logoColor=61DAFB)
![Expo](https://img.shields.io/badge/Expo-000020?style=flat&logo=expo&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat&logo=typescript&logoColor=white)
![Firebase](https://img.shields.io/badge/Firebase-FFCA28?style=flat&logo=firebase&logoColor=black)

## Funcionalidades

- Cursos em vídeo com módulos e aulas
- Questionários de nivelamento, pré e pós-conteúdo
- Certificados digitais automáticos
- Busca inteligente com IA (Google Gemini)
- Playlists e favoritos
- Histórico de questionários e progresso

## Tecnologias

- React Native + Expo SDK 54
- TypeScript
- Firebase (Auth, Firestore, Storage)
- Firebase AI Logic (Gemini 2.5 Flash)
- React Navigation
- expo-av

## Instalação

```bash
# Clone o repositório
git clone https://github.com/tiagolvjy/OncoPedAcademyMobile.git
cd OncoPedAcademyMobile

# Instale as dependências
npm install

# Inicie o projeto
npx expo start
```

Escaneie o QR Code com o aplicativo **Expo Go** (disponível na App Store e Google Play).

# [EN]

# OncoPed Academy — Mobile

A mobile educational platform for medical students and healthcare professionals focused on pediatric oncology. Developed as an undergraduate thesis (TCC) at Centro Universitário CESMAC, in partnership with the CITEC robotics lab.

![React Native](https://img.shields.io/badge/React_Native-20232A?style=flat&logo=react&logoColor=61DAFB)
![Expo](https://img.shields.io/badge/Expo-000020?style=flat&logo=expo&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat&logo=typescript&logoColor=white)
![Firebase](https://img.shields.io/badge/Firebase-FFCA28?style=flat&logo=firebase&logoColor=black)

## Features

- Video courses with structured modules and lessons
- Leveling, pre-content and post-content quizzes
- Automatic digital certificates
- AI-powered search (Google Gemini)
- Playlists and favorites
- Quiz history and progress tracking

## Tech Stack

- React Native + Expo SDK 54
- TypeScript
- Firebase (Auth, Firestore, Storage)
- Firebase AI Logic (Gemini 2.5 Flash)
- React Navigation
- expo-av

## Getting Started

```bash
# Clone the repository
git clone https://github.com/tiagolvjy/OncoPedAcademyMobile.git
cd OncoPedAcademyMobile

# Install dependencies
npm install

# Start the project
npx expo start
```

Scan the QR Code with the **Expo Go** app (available on App Store and Google Play).
